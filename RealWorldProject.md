# Real-World Data Project: Bank Loan Data Analysis and Risk Assessment

## Project Overview

This project focuses on analyzing bank loan data to understand the factors that influence loan approval decisions. Using Exploratory Data Analysis (EDA), the project identifies patterns, trends, and relationships between customer attributes such as income, age, credit score, and loan amount.

The goal is to apply data science techniques to a real-world finance dataset and generate meaningful business insights.

## Objectives

* Analyze customer financial data.
* Identify factors affecting loan approval.
* Perform data cleaning and preprocessing.
* Visualize trends and relationships using charts and graphs.
* Generate actionable insights from the dataset.

## Dataset Features

The dataset contains the following attributes:

* Customer_ID
* Age
* Income
* Loan_Amount
* Credit_Score
* Loan_Status

## Tools and Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Project Workflow

### 1. Data Collection

The loan dataset was loaded into Python using Pandas.

### 2. Data Cleaning

* Checked for missing values.
* Verified data types.
* Removed duplicates if present.

### 3. Exploratory Data Analysis (EDA)

Performed:

* Statistical summaries
* Distribution analysis
* Correlation analysis
* Loan approval analysis

### 4. Data Visualization

Created visualizations including:

* Income Distribution
* Credit Score Distribution
* Loan Approval Count Plot
* Income vs Loan Amount Scatter Plot
* Correlation Heatmap

## Key Findings

* Customers with higher credit scores were more likely to receive loan approval.
* Higher-income customers generally qualified for larger loan amounts.
* Credit score showed a strong relationship with loan approval status.
* Loan approval decisions were influenced by multiple financial factors.

## Conclusion

The analysis demonstrated how customer financial characteristics impact loan approval decisions. Credit score and income emerged as important indicators in the lending process. This project highlights the practical application of data analysis techniques in the finance domain.
